Installation

To start work with the template do next steps:

  1. Download and install latest stable version of node.js and npm.

  2. Install gulp: npm i gulp-cli.

  3. Open the folder with project in terminal (or cmd) and run: npm i.

  4. Edit *.scss or/and *.pug files

  5. Run gulp or gulp build to compile your changes.

Read documentation for sass and pug to know more.

New Page

To create new page you need to add title.pug to src/pug/ then add next default code:

extends ./layout/layout
  block content
    // add blocks here
  block scripts
    // add necessary scripts here

Add Block

Write variables, their values and add block (block content) after block content to add this block in the page. You can find used variables and examples of their values in comments lines of block.pug.

block content
  - var title = 'This title will be used in block.pug'
  include ./blocks/block

Add Script

Write path to script after block scripts to add it in your page.

block scripts
  script(src='assets/js/script.js')
